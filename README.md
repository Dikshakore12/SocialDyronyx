# DyronyxTech Website

A modern, responsive website for DyronyxTech – Digital Growth & Automation company specializing in social media strategy, Google review automation, website development, and business automation for Indian businesses.

## Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional design with smooth animations
- **Multi-page Navigation**: Home, Services, Product, Portfolio, Pricing, About, Blog, Contact
- **Interactive Elements**: Counter animations, smooth scrolling, mobile navigation
- **SEO Optimized**: Meta tags and structured content for better search visibility

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with modern features
- **JavaScript**: Interactive functionality and animations

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - this is a static website

### Running the Website

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the different sections using the menu

## Project Structure
